# Les différentes variétés  
Chocolate point  
SBI b

Les Sacrés de Birmanie chocolate point sont apparus en 1974 en Angleterre.

Les chocolate point sont légèrement moins contrastés que les seal point ; leurs extrémités sont d'un brun chaud, ne tirant pas sur le noir comme celles du seal. La fourrure du corps, dépourvue de reflets chez le très jeune animal, devient plus dorée avec l'âge.

Le chocolate point est comme le seal point une couleur non diluée. Il a donc les yeux assez foncés qui tirent parfois sur le violet.

La truffe est couleur chocolat au lait.

Les coussinets sont de préférence rose, sinon chocolat au lait ou les deux.

Cette couleur est le résultat d'une mutation qui remplace le pigment noir par du brun.
