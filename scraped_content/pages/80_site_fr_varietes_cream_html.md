# Les différentes variétés  
Cream point  
SBI e

Le cream point est la dilution du red point.

Les extrémités sont crème, un roux-brun moins chaud que chez le red point, poudre de riz. La couleur de son corps est d’un blanc cassé avec des reflets d'or. Le ton doit rester suffisamment froid pour bien se distinguer du red point. Avec cette couleur, l'éleveur doit s'armer de patience pour savoir si c'est un chat d'exposition ou de compagnie.

La truffe est rose.

Les coussinets sont également rose.
